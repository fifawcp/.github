# FIFA World Cup Prediction Platform

A web application for making predictions on FIFA World Cup matches, competing with friends in private boards, and tracking rankings.

## Repositories

| Repo | Description | Stack |
| ---- | ----------- | ----- |
| **[API](https://github.com/fifawcp/api)** | Backend REST API with passwordless authentication | Go, PostgreSQL, Redis |
| **[Web](https://github.com/fifawcp/web)** | React frontend for predictions and boards | Next, React, TypeScript, Tailwind |
